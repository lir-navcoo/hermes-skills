# A股高价值标的筛选与复盘框架

## 核心选股维度

### 一、技术面（权重40%）

**K线形态优先信号：**
- 锤子线：下影线≥实体2倍，收盘于高位 → 底部反转信号
- 吞没形态：第二根阳线完全包裹前一根阴线 → 看涨反转
- 启明之星：低位连续3根K线组合 → 底部反转
- 突破型阳线：放量突破前期高点/平台

**均线系统：**
- 多头排列：MA5>MA10>MA20>MA60，收盘站上所有均线
- 均线粘合后发散：盘整后放量选择方向
- 均线金叉：MA5上穿MA10且成交放量

**MACD：**
- 金叉（快线穿越慢线向上）+ 红柱放大
- 底背离：股价创新低但MACD未创新低 → 即将反弹

**成交量：**
- 上涨时放量（量价齐升）= 真实突破
- 连续放量且涨幅逐步扩大 = 主力介入
- 缩量上涨 = 可能是陷阱

**布林带：**
- 触及下轨后反弹 + 成交量放大
- 突破中轨 + 开口扩张

**涨停板战法：**
- 首板（第一涨停）：需结合板块效应判断
- 二板定龙头：连续涨停成妖概率增加
- 连板股：需分析市场情绪和板块持续性

### 二、基本面（权重30%）

**成长性指标：**
- 净利润增速：近2季度同比增速 > 20%
- 营收增速：近2季度同比增速 > 15%
-季度环比持续增长更佳

**盈利能力：**
- ROE（净资产收益率）：近年度 > 12% 为优质，> 20% 为优秀
- 毛利率：同比稳定或上升
- 净利率：同比稳定或上升

**估值：**
- PE（市盈率）：与行业均值对比，历史分位数
- PEG：PE/净利润增速，< 1 为低估
- PB（市净率）：周期性行业参考

**财务健康：**
- 经营现金流为正：现金流比利润更重要
- 资产负债率：< 60%（行业不同标准不同）
- 应收账款/存货：同比不异常增长

**股东结构：**
- 股东户数减少：筹码趋向集中
- 机构持仓增加：基金/QFII/社保增持
- 解禁压力：近期无大额解禁

**行业地位：**
- 细分行业龙头或寡头
- 具备定价权/护城河
- 市场占有率提升

### 三、资金面（权重30%）

**北向资金（沪深港通）：**
- 近3日累计净买入 > 1亿：外资关注
- 单日净买入超5亿：外资大幅买入
- 持续净流入且金额扩大：趋势性看好

**主力资金：**
- 超大单净流入排名：反映大资金动向
- 主力净流入占比：净流入/流通市值 > 0.5%
- 连续3日净流入：趋势性资金介入

**融资融券：**
- 融资余额较前日增长：杠杆资金看多
- 融券余额下降：看空力量减弱

**龙虎榜：**
- 机构专用席位买入：机构背书
- 知名游资席位：情绪资金介入
- 买入席位金额远超卖出：买方主导

---

### 四、热点与情绪

**政策面：**
- 重大政策出台次日：对应行业/板块有机会
- 产业规划/扶持政策：长期主题机会
- 货币/财政政策：影响整体市场风险偏好

**板块轮动：**
- 资金从高位板块向低位板块转移
- 每日成交额前5板块：主力资金聚集方向
- 概念题材：当前市场最强主线（如AI/半导体/新能源等）

**市场情绪指标：**
- 涨停家数：> 80家 = 情绪高涨
- 昨日涨停今日溢价率：反映打板资金赚钱效应
- 炸板率：炸板/涨停总数，越低越好

**龙头股效应：**
- 板块龙头不倒，跟风股有机会
- 龙头股高位横盘：资金在板块内轮动
- 龙头股跌停：警惕板块整体回调

## 早盘选股执行步骤

1. **市场概况扫描**：查看主要指数涨跌、成交额、涨跌家数比
2. **热点板块定位**：找到当日最强板块前3
3. **资金流向筛选**：从主力净流入榜找标的
4. **技术面验证**：逐一检查候选股K线形态、均线、MACD
5. **基本面核对**：用同花顺/东方财富查看财务数据
6. **精选5-10只**：按综合评分排序，给出推荐理由和风险
7. **输出清单**：代码、名称、推荐理由、风险提示、目标点位

## 下午复盘执行步骤

1. **指数复盘**：涨跌幅、成交额、贡献度
2. **板块复盘**：最强/最弱板块及原因
3. **推荐标的追踪**：逐个复盘推荐股票走势
4. **资金验证**：北向资金/主力是否验证逻辑
5. **错误反思**：判断错误的标的原因分析
6. **认知提升**：当天学到什么
7. **次日展望**：基于今日盘面预测明日方向

## 实时行情获取

### 腾讯行情API（Python）

```python
import urllib.request

# 获取多只股票实时价格
stocks = ['sz002097', 'sz002195', 'sh600633', 'sh600967']  # sz=深圳 sh=上海
url = 'https://qt.gtimg.cn/q=' + ','.join(stocks)
req = urllib.request.Request(url, headers={'User-Agent': 'Mozilla/5.0'})
response = urllib.request.urlopen(req, timeout=10)
data = response.read().decode('gbk')  # 必须用GBK解码

# 返回数据格式：v_sz002097="51~山河智能~002097~10.03~..."
# 字段说明：0=类型, 1=名称, 2=代码, 3=现价, 4=昨收, 5=今开, 6=成交量...
# 解析示例
for line in data.strip().split('\n'):
    parts = line.split('~')
    if len(parts) > 3:
        code = parts[2]
        name = parts[1]
        price = float(parts[3])
        change_pct = float(parts[32])  # 涨跌幅%
        print(f"{name}({code}): {price} ({change_pct:+.2f}%)")
```

### 注意事项
- 浏览器直接访问API会CORS拦截，必须用Python/后端
- 股票代码前缀：sz=深圳、sh=上海、us=美股
- 返回数据为GBK编码，非UTF-8
